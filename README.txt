This is the repository for all my code related to the full Run-II Stop2L analysis 
twiki: https://twiki.cern.ch/twiki/bin/viewauth/AtlasProtected/TtbarMET2L

Requirements:
- Setup LexTools (https://github.com/alexarmstrongvi/LexTools) and add it to your path
- Setup PlotTools (https://github.com/alexarmstrongvi/PlotTools)
- Setup susynt-write (https://gitlab.cern.ch/susynt/susynt-write)
    - This is needed to make the SusyNtuples for production or for local testing
- Setup susynt-read (https://gitlab.cern.ch/susynt/susynt-read)
    - This will setup the directory structure and other needed packages
- Then add into the source directory the following
    - Superflow (https://github.com/dantrim/Superflow)
